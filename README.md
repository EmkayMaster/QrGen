# qrgen - Modern QR Code Generator

A fast, free, and beautiful QR code generator built with modern web technologies. Create high-quality, customizable QR codes for any purpose.

![qrgen Screenshot](https://via.placeholder.com/800x400/6366f1/ffffff?text=qrgen+QR+Code+Generator)

## ✨ Features

- **🚀 Lightning Fast** - Client-side generation with no server delays
- **🎨 Fully Customizable** - Colors, sizes, error correction levels, and margins
- **📱 Multiple Content Types** - URLs, text, email, phone, SMS, WiFi, and GPS locations
- **🔒 Privacy First** - All processing happens in your browser
- **📱 Responsive Design** - Works perfectly on desktop, tablet, and mobile
- **⬇️ High Quality Output** - Generate crisp QR codes up to 800px resolution
- **🎯 Template System** - Pre-designed templates for common use cases

## 🎯 Supported QR Code Types

- **Website URLs** - Share links with automatic https:// formatting
- **Plain Text** - Any text content
- **Email Addresses** - Clickable mailto: links
- **Phone Numbers** - Clickable tel: links for instant dialing
- **SMS Messages** - Pre-formatted SMS links
- **WiFi Credentials** - Easy network sharing with security options
- **GPS Locations** - Shareable coordinates for navigation

## 📝 Usage Examples

### Basic URL QR Code
\`\`\`javascript
// Just enter: google.com
// Automatically becomes: https://google.com
\`\`\`

### WiFi Network Sharing
\`\`\`javascript
// Network: MyWiFi
// Password: mypassword123
// Security: WPA
// Generates: WIFI:T:WPA;S:MyWiFi;P:mypassword123;H:false;;
\`\`\`

### Contact Information
\`\`\`javascript
// Email: contact@example.com
// Becomes: mailto:contact@example.com
\`\`\`
